Decay Go Away

For my Junior Capstone project, I worked alongside four of my classmates to design and build a project relating to the field of agriculture. We ended the year with a device that would record environmental factors in real time. All of that data is then readily accessible on our web application where a user could make an account, register devices, and view the data relating to each device.

We achieved the hardware for this project with an Arduino UNO which read data using temperature, humidity, and ethylene sensors. That data was then pushed to an Arduino ESP32, which would sync that data with our AWS IoT Core instance.
